## 🚀 Cómo Usar

1. Clona el repositorio
2. Abre `index.html` en tu navegador
3. Personaliza las fotos en la sección de galería
4. Configura el formulario RSVP con tu Formspree ID

## 💡 Personalizaciones

Para agregar tus fotos:
1. Reemplaza las URLs placeholder en la sección de galería
2. Puedes subir tus fotos a GitHub usando GitHub Pages o usar URLs externas

Para configurar RSVP:
1. Crea una cuenta en [Formspree.io](https://formspree.io)
2. Reemplaza `YOUR_FORM_ID` en `index.html` y `script.js` con tu ID real
